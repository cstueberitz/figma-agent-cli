# Codex Profile

Purpose:
- Use Codex as a thin launch layer on top of the shared Figma backend.

How to Use:
- Read the repo root `AGENTS.md` first.
- Use `fig-start` for interactive launch.
- Use `fig-start --safe` for plugin-based sessions.

Practical Guidance:
- Use `node src/index.js doctor` and `node src/index.js smoke` for first-run checks.
- Keep helper files and follow-up edits aligned with the shared repo conventions instead of introducing Codex-only structure.
